### Purpose
Demonstrate **responsive and adaptive design patterns** in React using CSS media queries and custom hooks.

### Concept
- Use CSS for layout changes (`@media` queries).  
- Use `useMediaQuery()` hook for conditional rendering in React.  
- Combine both for seamless adaptive behavior.

### Run
npm install  
npm run dev

### Teaching Notes
- Resize the browser to switch between Mobile, Tablet, and Desktop modes.  
- Hooks allow conditional logic (e.g., render fewer items on mobile).  
- Works great with responsive grid libraries or styled-components.
